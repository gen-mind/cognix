func CollectionName ¶ 

func CollectionName(userID uuid.UUID, tenantID uuid.NullUUID) string

CollectionName returns the name of the collection based on the provided userID and tenantID. If the tenantID is valid, it formats the collection name as "tenant_{tenantID}" without "-" characters. Otherwise, it formats the collection name as "user_{userID}" without "-" characters. This function uses the constants CollectionTenant and CollectionUser for formatting. Example usage: 

func (c *Connector) CollectionName() string {
  return CollectionName(c.UserID, c.TenantID)
}
